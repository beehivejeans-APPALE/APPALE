# APPALE!

## Mission
日本語で話すだけで、世界に向けたクラウドファンディングページを完成できるプラットフォームを創る。

## Vision
「日本語でAIと対話するだけ」で、クラウドファンディングの企画からページ公開までが完了する世界を実現する。

## Why（解決する課題）
- クラファンの「言語化コスト」が高すぎる
- 良いアイデアが形になる前に止まる
- LP制作・ストーリーライティングが専門領域化している
- 成功パターンが属人化している

結果として、「やりたい人は多いが、実行できない構造」になっている。

## Concept
「日本語でAIに話すだけ」
「企画がそのままページになる」
「クラファン制作の民主化」

従来必要だった以下をAIが代替する：
- 企画書作成
- ストーリーライティング
- LP制作
- 構成設計
- コピーライティング

## Core UX Flow
1. ユーザーが日本語でAIに話す（例：「地域の古民家を再生したい」）
2. AIが質問しながら企画を構造化する（目的・ターゲット・ストーリー・リターン設計）
3. AIがクラファンページを自動生成する（タイトル・説明文・ストーリー・画像構成案・セクション設計）
4. そのまま公開可能な状態で出力する

## APPALE!の本質
単なる文章生成AIツールではなく、「クラファン制作プロセスそのものをUI化したプロダクト」である。
入力＝会話、出力＝クラファンページ、という構造変換ツール。

## Target Users
- 地方創生プロジェクト
- 個人クリエイター
- スタートアップ初期
- NPO / 社会活動
- 初めてクラファンをやる人

## Differentiation
- 日本語ネイティブ特化UX（会話型）
- ページ生成まで一気通貫
- クラファン専用設計（汎用AIではない）
- 成功パターンのテンプレ化が可能
- ChatGPTのような「文章生成AI」ではない
- NotionやLPツールのような「編集ツール」でもない

## Goal
- 誰でもクラファンを立ち上げられる世界
- 制作コストのほぼゼロ化
- 成功確率の底上げ
- クラファンのインフラ化

## Note
海外展開全般をサポートする事業（SELLPORT）は本プロジェクトとは別軸の構想であり、
APPALE!は「クラウドファンディングのページ生成」に領域を絞る。

## Future: APPALE!MALL（継続販売への発展）

現時点では将来構想であり、Phase 1〜2のスコープには含まない。クラファン生成の中核体験（企画→ページ公開）が安定してから着手する、Phase 3以降の構想。

### Why（存在理由）
クラファンの多くは「単発の資金調達」で終わり、成功後の関係性・販売チャネルが途切れてしまう。せっかく集まった支援者・ファンとの接点を、継続的な販売機会に転換できていない。

### Concept
クラファンページが、成功後もそのまま「ミニ・オンラインストア」に変化する。新しいツールへの乗り換えや再構築は不要——同じページ・同じストーリー・同じ支援者基盤の上に、継続販売の機能が追加されるだけ。

### Core Flow（想定）
1. クラファンが終了・成功する
2. APPALE!が「MALLへの移行」を提案する
3. 既存のストーリー・画像・リターン設計を土台に、通常販売用の商品ページへ自動変換
4. 支援者はそのまま継続購入者になれる

### Differentiation
- 単なるECカート追加ではなく、「物語ごと販売を継続する」設計
- BASEやShopifyのような汎用ECではなく、クラファン発の商品に特化
- 初期構築コストがほぼゼロ（既存ページの資産を再利用）

### Phase（発展タイミング）
Phase 1〜2のスコープには含まない。クラファン生成の中核体験（企画→ページ公開）が安定してから着手する、Phase 3以降の構想。

## Future: 認証方式の拡張
現在はマジックリンク（メールのみ）方式だが、将来的にメールアドレス＋パスワード方式も併用できるようにする。
Supabase Authで両方式を併用可能なため、大きな作り直しは不要な想定。

## Future: クラファン公開中の伴走AI
クラウドファンディング公開後も、ユーザーがいつでも相談できるAI伴走役を提供する。
このAIは特にマーケティング視点でのアドバイスを得意とする性格付けとする。

想定する役割：
- 支援状況（進捗・残り日数）を踏まえた改善提案（マーケティング視点）
- 追加リターン設計、SNS発信内容、価格戦略などの相談対応
- 支援者からの質問対応のサポート

これにより、APPALE!は以下の3フェーズすべてでAIが伴走するプロダクトになる：
1. 立ち上げ前: 日本語対話 → クラファンページ生成
2. 運営中: マーケティング視点の伴走AI（本セクション）
3. 成功後: APPALE!MALLで継続販売

現時点では将来構想であり、Phase 1〜2の実装スコープには含まない。
